Pylon

Multi-agent platform that helps career changers find and apply to jobs strategically using AI-powered research, personalized outreach, and intelligent pipeline orchestration.

6 Specialized Agents · Actor-Critic Workflows · Swarm Research

🔍 Smart Discovery

Input free-text interests like "I love European football, find DS/ML companies" and get 10-15 curated company matches with relevance scores.

🧠 Deep Research

Each company is profiled with R&D approach, tech stack, ML use cases, funding stage, culture, and hiring signals.

📊 Skill Gap Analysis

Compare your skills against each company's tech stack. Get alignment scores and actionable learning paths.

📧 Personalized Outreach

Auto-draft cold emails personalized to each company's projects, culture, and the right decision-maker.

📝 Tailored Resumes

Resume summaries and bullet points customized per company, emphasizing relevant experience and skills.

📈 Excel Reports

Multi-tab Excel exports with companies, skills analysis, contacts, and outreach drafts — all in one file.

Architecture

Router Orchestration

7-priority intent classification dispatches to the right agents and workflows automatically.

Actor-Critic Loops

Search plans and outreach drafts go through max 3 refinement cycles for quality assurance.

Swarm Research

Parallel multi-company research using ThreadPoolExecutor for fast, comprehensive analysis.

Stable-Anchor Memory

Persistent knowledge in markdown files with safe append-after-header editing patterns.

Pydantic Contracts

Every agent returns a RouterContract with status, confidence, and knowledge updates.

Live WebSocket

Real-time pipeline progress via FastAPI WebSocket — see each step as it completes.
