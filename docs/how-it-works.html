Search Modes

Choose a search mode before running your query to focus the agents on the right type of opportunities.

General Mode

  • Searches for any type of role
  • Broad industry and company coverage
  • Generic skill analysis and outreach
  • Best for exploring opportunities widely

DS / ML Mode

  • Targets data science and ML roles specifically
  • Finds DS/ML teams at companies in ANY industry
  • Emphasizes ML frameworks, data tools, and AI infrastructure
  • Contacts data science leadership specifically

DSPy Optimization Layer

When DSPY_ENABLED=true, every agent runs through DSPy modules instead of direct LLM calls. DSPy optimizes prompts automatically using scored metrics per agent. The Actor-Critic workflow (Planner + Critic) also has DSPy modules for iterative plan refinement.
